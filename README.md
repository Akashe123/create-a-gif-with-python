# Create a GIF with Python

This project creates an animated GIF from multiple images using Python and the Pillow library.

## Features

- Reads multiple images
- Combines them into a GIF
- Adjustable animation speed
- Infinite looping GIF

## Requirements

- Python 3.x
- Pillow

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python create_gif.py
```
